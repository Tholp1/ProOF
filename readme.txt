proOF is a config aimed at making open fortress more competitively viable through weapon rebalances, the reintroduction of armour and more in order to make a more fast paced and fun experience

put the contents of this folder into steamapps/sourcemods/open_fortress/custom
make sure the directory is ProOF/(files), not ProOF/ProOF/(files)

to disable ProOF if you want to play casual dm, move everything in the folder to the '_disable' folder

glhf!
